## 🚀 Featured Projects

### 🔸 [Airtel AI Customer Service Chatbot](https://airtelcustomerservicebot-zewtnedlyhbfaefe8lmrfh.streamlit.app/)
- 🌐 **LLM-powered chatbot** trained on Airtel-style telecom data
- ⚡ Uses **LangChain**, **Groq LLaMA3-70B**, **FAISS**, and **Streamlit**
- 💬 Handles billing, SIM, recharge, plans, and customer queries dynamically

### 🔹 [Driver Drowsiness Detection System](https://github.com/rushikram/Drivers-Drowsiness-DETECTION-system)
- 🧠 Real-time eye-closure detection with **OpenCV** and **Dlib**
- 🚨 Alerts driver when prolonged drowsiness is detected
- 📹 Uses webcam feed for facial landmark tracking

### 🔹 [AI-Powered Blog Generator](https://github.com/rushikram/Blog-Generator---Using-llama-2-7B)
- ✍️ Auto-generates blogs using **LLaMA2-7B**, **LangChain PromptTemplate**, and **Streamlit**
- 🎯 Inputs: topic, tone, word count, and audience preferences
